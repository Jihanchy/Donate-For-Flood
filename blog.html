Important FAQ

What is the Document Object Model?

The document Object Model is a programming articulation for web document.It shows the page that by programming one can change the document style, content.It shows the documents as object and nodes, that's why programs can connect with the web page..

How do you select an element from the DOM?

DOM shows the web page as like object and nodes, that's why I can select an element from the DOM by several ways.Here it is:
By using an id: example- getElementById()
By using class: example- getElementByClassName()
By using selector: example- querySelectorAll()/querySelector()

What is event delegation in the context of DOM, why is it useful?

Event delegation is one of event bubbling method. It is an event handler that allows you to handle the event at the top in DOM tree to where the event was first time declared.The event delegation is very useful for large content.Because in the web page we have to add many event listener. So, by event delegation we use one event listener that allows you connect a single event listener to the parent div.Consequently, we have less code in the document.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an elements attributes by using getAttribute, setAttribute, removeAttribute.
Besides this, by using classList.add, classList.remove, templateString, document.style.color etc i can manipulate the styles in DOM.
